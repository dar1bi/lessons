АТРИБУТ 'CLASS'

Атрибут 'class', в основному, використовується для вказівки на клас в каскадній таблиці стилів (CSS). Тим не менш, він також може бути використаний в JavaScript (через HTML DOM), щоб внести зміни в HTML елемент з зазначеним класом.

Правила написання імені класу:

  1. ім'я довільне, але логічне - описує цей об'єкт
  2. тільки латиниця (eng)
  3. тільки маленькі (прописні) літери
  4. пробіли у імені класу (коли в імені більше одного слова) замінюємо на "-" (дефіс)

АТРИБУТ 'ID'

Атрибут 'id' задає ідентифікатор - унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти. Ідентифікатор в коді документа повинен бути в єдиному екземплярі, іншими словами, зустрічатися тільки один раз.

  1. ім'я довільне, але логічне - описує цей об'єкт
  2. тільки латиниця (eng)
  3. тільки маленькі (прописні) літери
  4. пробіли у імені класу (коли в імені більше одного слова) замінюємо на "-" (дефіс)

Опишемо деякі інші додаткові атрибути, які часто використовуються в тегах:

  • title - довільний текст підказки, яка з'являється при наведенні на об'єкт

ПІДКЛЮЧЕННЯ CSS ТА ЙОГО СИНТАКСИС

Каскадні таблиці стилів (англ. Cascading Style Sheets) — спеціальна мова, що використовується для запису оформлення сторінок, написаних мовами розмітки даних. Концепція стилів працює так - текст спочатку виводиться, а потім форматується, за допомогою CSS стилів. Cтилі дають змогу розмежувати вміст веб сторінки від її оформлення.

Існує 3 основних варіанти підключення каскадних таблиць стилів. А саме:

1 варіант: За допомогою тега STYLE

2 варіант: За допомогою атрибуту STYLE

3 варіант (Основний!): Підключення окремого файлу

В заданому файлі, який ми створили та підключили, будемо прописувати всі наші стилі.
