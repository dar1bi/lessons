ЩО ТАКЕ HTML?

HTML (англ. HyperText Markup Language — мова розмітки гіпертексту) — стандартизована мова розмітки документів для перегляду веб-сторінок у браузері. Веб-браузери отримують HTML документ від сервера за протоколами HTTP/HTTPS або відкривають з локального диска, далі інтерпретують код в інтерфейс, який відображатиметься на екрані монітора.

HTML документ буде складатися з деякої групи елементів, де кожен елемент буде визначатися (починатися та закінчуватися) певним тегом (для деяких елементів кінцевий тег не є обов'язвовим). Тобто, тег — це назва елементу, записана у кутових дужках (< >). Кожен HTML тег має свою унікальну назву з визначеним синтаксисом, яка записується латинськими літерами і чутлива до регістру.

Будівельні теги

Всі будівельні теги є блочними! Отже, до цієї категорії виділяємо наступні теги:

  1. DIV - тег без призначення (<div></div>)
  2. HEADER - шапка сайту/блоку (<header></header>)
  3. FOOTER - підвал сайту (<footer></footer>)
  4. MAIN - головна частина сторінки (<main></main>)

Теги для контенту

Контент в html документі - це, наприклад:

  1. посилання
  2. зображення
  3. відео
  4. аудіо
  5. текст
  6. і т.д.

І для кожного типу контенту має бути свій тег (по семантиці). Отже, виділяють наступні теги для контенту:

P - параграф тексту (блочний). Приклад:
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Illo esse veritatis eaque voluptas quod reiciendis quasi accusantium, laboriosam facilis aliquid quia nulla eveniet repellat expedita similique minima unde nam quibusdam?</p>

A - посилання (рядковий). Приклад:
<a target="_blank" href="https://google.com">Посилання на Google</a>

IMG - додавання зображення (рядковий). Для семантики та доступності потрібно заповнювати атрибут alt. Приклад:
<img src="img/01.png" alt="Будівництво">

BR - переревання рядка (МАЙЖЕ блочний). Написання:
<br>

SPAN - виділення частини рядка (рядковий). Написання:
<span></span>

MARK - виділяє/підсвічує частину рядка (рядковий). Написання:
<mark></mark>

UL - маркований список (блочний).
LI - елемент списку (блочний). Написання:
<ul>
  <li>Пункт</li>
</ul>

OL - нумерований список (блочний). Додаткові атрибути тегу. Написання:
<ol>
  <li>Пункт</li>
</ol>

H* - заголовок (блочний). Заголовок h буває 6 рівнів, а саме:
<h1>Я заголовок 1</h1>
<h2>Я заголовок 2</h2>
<h3>Я заголовок 3</h3>
<h4>Я заголовок 4</h4>
<h5>Я заголовок 5</h5>
<h6>Я заголовок 6</h6>


ТИПИ ТЕГІВ:

  • Блочні
  • Рядкові

Блочний тег займатиме усю ширину батьківсього елементу. Висоту - згідно контенту. Наступний, після блочного елементу, об'єкт, буде виведений з нового рядка.
Рядковий тег займатиме ширину згідно контенту. Висоту - згідно контенту. Наступний, після рядкового елементу, рядковий об'єкт, буде намагатись розміститись з ним у рядок.


ДОДАТОК:

Гарячі клавіші (VS Code, мій список, який буде доповнюватися в процессі навчання)

  • Cmd + ?/ - коментар
  • Cmd + Z - скасування (назад)
    • Cmd + Shift + Z - скасування (вперед)
  • Alt + Z - перенесення тексту по ширині редактора (вкл/викл)
  • Cmd + «+» - збільшення всіх елементів програми
    • Cmd + «-» - зменшення всіх елементів програми
    • Cmd + «0» - скинути попередні налаштування
  • Cmd + , - виклик параметрів програми
  • Alt + W - обернути виділене в тег (за замовчуванням в <p>, можна відразу змінювати)
  • Cmd + Alt + «↑/↓» - виділення та робота відразу з декількома рядками (наприклад, для роботи відразу з усіма <li>)
  • Shift + Alt + «↑/↓» - копіювання поточного рядка (вверх або донизу)
